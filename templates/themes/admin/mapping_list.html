What is Theme Mapping?

Theme Mapping connects the visual elements of your uploaded HTML theme with dynamic database fields belonging to user portfolios.

You can create multiple mapping profiles (e.g., experimental versions or specific layouts) and choose one to be the Active Profile. The active profile is used by the rendering engine to compile and serve live portfolios.
