Frequently Asked Questions

How does the AI Resume parsing work?

We use Google Gemini API layout mapping. It reads text, divides sections intelligently, sanitizes HTML, and populates the portfolio structure automatically.

Do I need to purchase a hosting plan?

No! Hosting is included for free. If you're using our GitHub integration, your site is hosted directly on GitHub's secure servers.

Can I map subdomains like portfolio.myname.com?

Yes, you can register any custom subdomain or root level domains in the workspace setup with Let's Encrypt SSL applied.
